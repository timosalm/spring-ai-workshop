Welcome to this hands-on workshop to learn about **Spring AI**. 

## What You'll Build

Throughout this workshop, you'll incrementally build a fully functional AI-powered support assistant that can:

- Answer questions about Tanzu Spring support offerings
- Parse and classify support tickets from natural language
- Search a knowledge base using RAG (Retrieval-Augmented Generation)
- Use tools to fetch real-time information and create tickets

## Workshop Modules

### Required Modules
