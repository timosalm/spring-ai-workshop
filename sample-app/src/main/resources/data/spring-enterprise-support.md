# Tanzu Spring Support

## Overview

Tanzu Spring is Broadcom's commercial offering that provides enterprise-grade support for Spring applications. It is designed for organizations that rely on Spring Framework and Spring Boot for mission-critical applications.

## Support Tiers

### Standard Support
- Business hours support (8x5)
- Response times: P1 (4 hours), P2 (8 hours), P3 (24 hours), P4 (48 hours)
- Access to knowledge base and documentation
- Email and web portal support

### Premium Support
- 24/7 support coverage
- Response times: P1 (1 hour), P2 (4 hours), P3 (8 hours), P4 (24 hours)
- Named Technical Account Manager (TAM)
- Quarterly business reviews
- Phone, email, and web portal support

### Mission Critical Support
- 24/7 dedicated support team
- Response times: P1 (15 minutes), P2 (1 hour), P3 (4 hours), P4 (8 hours)
- Dedicated Technical Account Manager
- Proactive monitoring and health checks
- On-site support available

## CVE Response Program

Tanzu Spring customers receive priority access to security patches:

1. **Early Notification**: Customers are notified of critical CVEs before public disclosure
2. **Priority Patches**: Security fixes are available to enterprise customers first
3. **Backported Fixes**: Security patches are backported to LTS versions
4. **Remediation Guidance**: Detailed instructions for applying patches

### Recent CVE Examples
- CVE-2024-38816: Spring Framework path traversal vulnerability
- CVE-2024-38809: Spring Framework denial of service vulnerability
- CVE-2024-22262: Spring Framework URL parsing vulnerability

## Long-Term Support (LTS)

Tanzu Spring provides extended support for Spring Boot versions:

| Version | Community EOL | Enterprise EOL |
|---------|--------------|----------------|
| Spring Boot 3.2.x | Nov 2024 | Nov 2026 |
| Spring Boot 3.1.x | May 2024 | May 2026 |
| Spring Boot 2.7.x | Nov 2023 | Nov 2025 |

## Getting Support

### Support Portal
Access the support portal at https://support.broadcom.com/example

### Creating a Ticket
1. Log in to the support portal
2. Select "Create New Case"
3. Choose product: Tanzu Spring
4. Select severity level based on impact
5. Provide detailed description and logs

### Severity Definitions
- **P1 (Critical)**: Production system down, no workaround
- **P2 (High)**: Major functionality impacted, workaround available
- **P3 (Medium)**: Minor functionality impacted
- **P4 (Low)**: General questions or enhancement requests
